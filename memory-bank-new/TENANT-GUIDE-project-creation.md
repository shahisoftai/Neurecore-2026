# Tenant Guide: Creating a New Project via Hermes Chat

**Last updated:** 2026-07-18

---

## How Projects Are Created

In NeureCore, you create projects by **chatting with your AI assistant (Hermes)** — not through a form or wizard.

The chat is accessible from:
1. **The AI prompt on your home page** — type directly in the input at the top of the screen
2. **The AI Chat button** — the floating button in the bottom-right corner

---

## How to Start a Chat

### Option 1: Home Page Prompt

Go to your **Home page** (`/home`). At the top you'll see:

- A greeting with your name
- An AI prompt input: *"Ask any agent or department…"*
- Quick suggestion chips: *"How's our pipeline this week?"*, *"Show pending approvals"*, etc.

**Click the input and type your project request.**

### Option 2: Floating AI Chat Button

Click the **AI Chat button** (bottom-right corner) to open the full chat panel. You can type any request there.

---

## Creating a Project via Chat

Type something like this in the chat:

- *"We just won the Acme Corp audit engagement — Q3 deadline, fixed fee $75k"*
- *"Start a new regulatory examination project for FinanceHub"*
- *"I need to launch a product by December, $50k budget"*
- *"New client: Pinnacle Group, they need help with annual tax filing"*

You can also drop files, paste emails, or attach documents — Hermes will interpret them.

---

## What Happens Next

Hermes will:

1. **Recognize your intent** — understands you're starting a project
2. **Infer the project type** — matches against 150+ industry templates
3. **Pre-fill information** — pulls what it can from your message/attachments
4. **Ask only for genuine gaps** — not a long form, just what's missing
5. **Propose a draft** — shows you what it will create for confirmation

Example proposal from Hermes:
> *"I'll create a Regulatory Examination project for Acme Corp, budget $75k, deadline Q3. I still need: examination scope, regulator contact. Does this look right?"*

**Confirm or correct** — then Hermes creates the project.

---

## What Happens After Creation

Once confirmed, NeureCore automatically:

| Feature | Description |
|---|---|
| **AI Employees Spawn** | Based on project type, relevant AI employees are assigned |
| **Chief of Staff Assigned** | Your AI Chief of Staff monitors the project |
| **Goals Suggested** | Project-type-specific goals are auto-generated |
| **Tasks Decomposed** | Goals are broken into actionable tasks |
| **Project Memory Seeded** | Initial context is populated for AI awareness |

---

## Managing Projects via Chat

You can also manage projects through the same chat:

- *"What's the health of the Acme project?"*
- *"Add a task to review the proposal by Friday"*
- *"What goals are at risk in the Q4 launch?"*
- *"Show me all active projects"*
- *"Create a new task: follow up with the client"*

---

## Project Status Lifecycle

```
Lead → Proposal Sent → Won / Lost
Won → Active ⇄ On Hold → Review → Completed → Archived
Lost → Archived
```

---

## Fallback: Manual Wizard

If Hermes cannot interpret your request, use the manual wizard:

**https://hq.neurecore.com/projects/new**

This is the explicit fallback — chat is always preferred.

---

## Quick Tips

- **Be conversational** — Hermes understands natural language, not form fields
- **Drop supporting files** — emails, contracts, proposals are all interpreted
- **Confirm before acting** — Hermes always proposes before creating anything
- **Ask anything** — about any project, task, goal, or decision
- **Check health** — ask *"how is [project] doing?"* for an AI health summary
