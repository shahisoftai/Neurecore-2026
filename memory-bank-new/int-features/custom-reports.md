# Custom Reports

## Overview
Allow tenants to build, schedule, and export custom reports combining data from AI employee activities, task completions, costs, and operational metrics.

## Category
ANALYTICS

## Backend Status
- ⚠️ **Partial** — `ReportsTool` generates HTML reports from Prisma data
- Report types: task_summary, cost_summary, agent_workload, pipeline_overview
- PDF export via Google Drive export API
- No standalone report builder or scheduling system
- No saved report templates in the database

## Tenant Frontend Status
- ❌ **No tenant report builder UI**
- Reports are generated by agents in chat, not through a standalone interface

## Admin Frontend Status
- ✅ **Admin has `/strategy` page** for financial forecasting scenarios
- Not a general-purpose report builder

## AI Employee Integration
- ✅ **ReportsTool** generates and saves reports to Google Drive
- Agents can generate reports on demand and email them
- Limited to 4 hardcoded report types — no custom report definition

## Package/Tier Integration
- Key: `custom_reports`
- Referenced in accounting packages

## Implementation Gaps
- Report template builder UI (drag-and-drop metric selection)
- Scheduled report delivery (daily/weekly/monthly via email)
- Saved report templates in database
- Multiple export formats (PDF, CSV, Excel, HTML)
- Report sharing and collaboration
- Parameterized reports (filter by department, agent, date range)
- Agent tool for custom report creation (beyond the 4 hardcoded types)
